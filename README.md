# 家装预算快速估算器

一个纯粹的 HTML/CSS/JS 前端工具，用于快速估算家装预算。

## 功能

- **柜子计算器**：按房间分别输入尺寸，支持多组柜子，计算投影面积和预估价格
- **窗户计算器**：按房间分别输入尺寸，支持多组窗户，含窗扇数量计算，计算窗户面积和预估价格
- **统一参数**：支持批量设置统一单价和高度
- **房间明细**：计算后显示每个房间的面积和价格明细
- **数据导出**：一键导出完整的预算报告（TXT 格式）
- **表达式输入**：在输入框中直接输入算式（如 `2000+300`），按回车自动计算结果
- **自动配色**：计算后的房间有绿色标识，修改数据后自动清除

## 技术栈

- Tailwind CSS（CDN 引入）
- 纯前端，无后端依赖
- 单 HTML 文件，零配置
